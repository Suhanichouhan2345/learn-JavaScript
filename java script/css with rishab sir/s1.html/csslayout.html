CSS

CSS is the acronym for "Cascading Style Sheet". It's a style sheet language used for describing the presentation of a document written in a markup language like HTML. CSS helps the web developers to control the layout and other visual aspects of the web pages.

Types of CSS

1. Inline CSS :- Written directly inside an HTML element
2. Internal CSS :- Written inside the style section of an HTML page
3. External CSS :- Written in a separate CSS file

Selectors :- They decide which HTML element the CSS will apply to.

1. CSS selectors are used to choose HTML elements on which styles are applied.
2. A selector tells the browser which element should be styled.

Types of CSS Selectors

1. Element Selector :- Selects elements by their tag name.
2. Class Selector :- Selects elements using a class name. 3. ID Selector :- Selects an element using a unique ID.
4. Group Selector :- Used to apply the same style to multiple elements at once. 5. Universal Selector :- Selects all elements on the webpage.

CSS Version

1. CSS1 - Basic styling (text, color, margin, background) 2. CSS2 - Positioning and layout control
3. CSS2.1 - Browser compatibility improved 4. CSS3 - Animation, responsive & advanced design
5. CSS4 - New features added to CSS3 (ongoing)
